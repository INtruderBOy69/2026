# Happy New Year 2026

A visually stunning, pure CSS animated webpage celebrating the New Year 2026.

![Demo](Demo.png)

## Features

- **Year Transition Animation** - Smooth animated transition from 2025 to 2026
- **Balloon Animation** - A floating balloon carries the "6" into position
- **Fireworks Effect** - Celebratory fireworks burst across the screen
- **Pure CSS** - No JavaScript required; all animations built with CSS keyframes
- **Responsive Typography** - Uses Montserrat and Pacifico Google Fonts

## Technologies

- HTML5
- CSS3 (Animations, Flexbox, Keyframes)

## Usage

Simply open `index.html` in any modern web browser to view the animation.

```bash
open index.html
```
